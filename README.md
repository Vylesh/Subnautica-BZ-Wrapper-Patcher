An Personal Wrapper maker script for Subnautica BZ with Submersed BZ VR for Meta Quest devices 

This Patcher script will add "Real" part to original game and data folder with creates and wrapper exe named SubnauticaZero.exe
wrapper will make the game run without its default argument like "-silent-crashes -vrmode none" and will use steamvr arguments
